# Basirah — بصيرة
> *"Those who spend their wealth by night and by day, secretly and openly —*
> *their reward is with their Lord, and there will be no fear upon them, nor will they grieve."*
> — Al-Baqarah 2:274

**Basirah** (بصيرة) means *insight* — seeing truth with clarity. It turns humanitarian data into understanding, and understanding into action.

---

## The Problem

Humanitarian crises only get funded when they go viral. Millions suffer in silence because they never trend. Basirah was built to fix that.

---

## What It Does

**One flow:** `Explore → Understand → Donate → See Your Impact`

| | |
|---|---|
| 🌍 **3D Crisis Globe** | Explore 12 active crisis zones plotted on an interactive globe, color-coded by severity |
| 📋 **Crisis Intelligence** | Real-time updates, funding gaps, and verified aid organizations per zone |
| 💸 **Zakat & Sadaqah** | Choose your intention, select an amount, donate in CAD |
| ✨ **Impact Beam** | Watch a beam of light travel from your location to the crisis zone when you give |

---

## Getting Started

```bash
git clone https://github.com/your-username/basirah.git
cd basirah
npm install
```

Create a `.env` file:

```env
VITE_MAPBOX_TOKEN=your_mapbox_token
VITE_GOOGLE_CLIENT_ID=your_google_client_id
```

```bash
npm run dev
# → http://localhost:8080
```
